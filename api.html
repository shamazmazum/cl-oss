Here is an API reference.
dsp-device
DSP device. Not to be instaniated
  • name
    dsp device filename
  • stream
    Underlaying stream
  • file-desc
    File descriptor of the underlaying stream
  • sample-format
    Sample format understood by OSS
  • channels
    Number of audio channels
  • sample-rate
    Sample rate
dsp-device-output
Class for output to DSP device
    dsp-device-input
    Class for input from DSP device
      with-dsp-device((device class &rest args) &body body)
      Bounds device to open dsp device of class class. Closes the device when control leaves the body. args are passed to make-instance on creation of the device.
