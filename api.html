      with-dsp-device((device class &rest args) &body body)
      Bounds device to open dsp device of class class. Closes the device when control leaves the body. args are passed to make-instance on creation of the device.
      device-native-formats(device-name &optional (direction output))

      Return a list of sample formats natively supported by a device. Direction can be either :input or :output.

      NB: This function may return incorrect results, for example on FreeBSD when hw.snd.report_soft_formats sysctl is set to 1.
